# Hardware

- `microgroove_labels_v6_preview.png` — the v6 keymap / keycap label reference
  (one key = one function; orange = hold, green = sampling). This doubles as the
  quickest explanation of the whole layout.
- `shell/` — printable STLs for the Microgroove shell, also on MakerWorld & Printables
- `microgroove_labels_v6.svg` — the print-ready label sheet. Print with a 0.2 mm
  nozzle for clean fine text; the label top and keys use a multi-color plate.
  v6 is current for the polyphonic firmware — no reprint needed if you already have v6.

Print settings, assembly photos, and the label application guide live on the
MakerWorld listing.
